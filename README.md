**Project Overview**
This project focuses on analyzing employee bank transactions to identify policy deviations, conflict of interest, unusual financial behavior, and audit red flags using SQL.
It simulates a real corporate audit environment, similar to what MIS and Internal Audit teams perform.
**Objective**
- Detect high-value and frequent transactions
- Identify employee-client financial relationships
- Analyze loan exposure across NBFCs
- Support compliance and internal audit teams
**Tools Used**
  MySQL
  GitHub
**Key SQL Concept**
- JOINs
- Aggregate Functions
- CASE Statements
- Subqueries
- GROUP BY & HAVING
**Business Impact**
- Reduced manual audit effort
- Early risk identification
- Data-driven compliance reporting
